## Getting Started

### Prerequisites

You need to have a browser and this tools installed:

- Git
- Ruby
- npm
- node
- yarn

### Setup

To get a local copy up and running follow these simple example steps:

```
$ git clone https://github.com/Shubh-Dev/hello-rails-react.git
```

```
$ cd hello-rails-react
```

```
$ bundle install

$ yarn install

$ npm install

$ rails db:create

$ rails db:migrate

```

### Usage

To run the server:

```
$ ./bin/dev
```
